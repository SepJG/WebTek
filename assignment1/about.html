Questions

  1. What does the server in the client server architecture do?

    The server is software running on a computer, responding to requests from the client. The server retrieves and transmits file to the client.

  2. What does the client do?

    In a web architecture, your browser (e.g. Chrome, Safari, Firefox) is the client. The browsers locates and displays information from the web. Communication between the client and server is done by an agreed transmission language, or protocol, e.g. HyperText Transfer Protocol(HTTP). The client requests and recieves services(e.g. files, applications) from the server

  3. What is an IP address?

    IP stands for Internet Protocol. IP address is a unique adress or unique string of characters that identifies a device on the Internet or a local network. Internet protocol is the set of rules governing the format of data sent via the internet or local network.

  4. What is DNS?

    DNS stands for Domain Name Server. DNS translates domain names into IP addresses, which allow browsers to get to websites and other internet resources. The DNS machines are responsible for providing the IP mapping and for the upkeep of the database as new machines and IPs are added the the Internet.

  5. What is the difference between the Internet and WWW?

    The Internet is the "network of networks", formed by connection of millions of computers. But the world wide web(WWW) is a service or application on the Internet that stores data and information and allows us to access it.

  6. What is HTML?

    HTML(HyperText Markup Language) is a stnadard markup language that is used to declare what should exist on a web page. HTML describes the structure of a page.
